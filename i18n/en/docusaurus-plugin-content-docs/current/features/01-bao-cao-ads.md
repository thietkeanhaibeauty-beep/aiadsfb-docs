# View Ads Report

View detailed advertising performance reports with key metrics in real-time.

---

## Description

The View Ads Report feature allows you to:

- View spend, results, phone numbers, revenue, ROI reports
- Reports by day, week, month
- Compare performance with previous periods
- Sales reports (leads, appointments, conversions)

---

## Common Commands

| Command | Description |
|---------|-------------|
| `Today's report` | View report for current day |
| `Today's sales report` | View leads, contacts, appointments, conversions |
| `Weekly report` | Summary of last 7 days |
| `Monthly report` | Full month summary |
| `Compare to yesterday?` | Compare with previous day |

## Key Metrics

| Metric | Meaning |
|--------|---------|
| **Spend** | Total amount spent on advertising |
| **Results** | Number of interactions/messages/leads |
| **Phone leads** | Phone numbers collected |
| **Revenue** | Revenue from ad customers |
| **ROI** | Return on Investment (Revenue / Spend) |
| **CPR** | Cost per Result (Spend / Results) |

Check your reports daily to monitor performance and make timely adjustments!
