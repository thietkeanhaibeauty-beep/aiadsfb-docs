# View Ad Campaigns

View all campaigns, turn on/off with toggle or chat with AI.

---

## Description

Easy campaign management:

- View list of all campaigns
- Display metrics: Spend, Results, CPR
- Quick on/off with toggle
- Or control via chat

---

## Common Commands

| Command | Description |
|---------|-------------|
| `View all campaigns` | Display campaign list |
| `Turn off campaign [name]` | Pause campaign |
| `Turn on campaign [name]` | Reactivate campaign |
