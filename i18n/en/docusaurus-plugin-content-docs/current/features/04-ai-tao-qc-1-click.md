# AI 1-Click Ad Creation

Create a complete campaign with just one simple command.

---

## Description

AI will automatically:

- Create Campaign with appropriate name
- Set up optimal budget
- Create Ad Sets
- Publish and run immediately

---

## Commands

```
Create message ad for Spa
```

Or more specific:

```
Create message campaign for laser hair removal service, budget $20/day
```

---

## Example

```
You: Create message ad for Spa

AI: Creating campaign...

AI: Done!
Camp: Spa Premium
Budget: $20/day
Ad Sets: 3

AI: Campaign is running!
```

---

## Supported Objectives

| Objective | Example Command |
|-----------|-----------------|
| Messages | `Create message ad...` |
| Leads | `Create lead gen ad...` |
| Website | `Create website traffic ad...` |
| Sales | `Create sales ad...` |

---

:::tip Tip
You can add details like budget, age, location for AI to create more accurately!
:::
