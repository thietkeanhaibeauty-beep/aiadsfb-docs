# Post Advertising

Paste a Facebook post link (Reels/Post) and AI will automatically create an ad for you.

---

## Description

This feature helps you quickly create ads from existing posts on your fanpage:

- Paste Facebook post link â†’ AI auto-creates campaign
- Use pre-saved targeting templates
- Complete in just seconds

---

## How to Use

### Step 1: Preparation

First, you need to create a **Targeting Template** with a keyword starting with `@#`.

> Don't know how? [See Targeting Template guide](/features/mau-nham-muc-tieu)

**Example keywords:** `@#spa`, `@#skincare`, `@#microblading`

---

### Step 2: Send Command to AI

Enter template keyword with post link:

```
@#TemplateName
https://www.facebook.com/reel/13531019863575924
```

## Important Notes

:::warning Requirements
- Post must be **public** and have "Send Message" button
- You must be the Page manager containing the post
- Targeting template must be created beforehand with `@#` keyword
:::

:::tip Pro Tip
Using **Reels** usually delivers better performance than regular posts!
:::
