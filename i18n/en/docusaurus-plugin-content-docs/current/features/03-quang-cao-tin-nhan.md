# Message Advertising

Create Facebook message ads with auto-reply scripts, greetings, and suggested questions.

---

## Description

Message Ads help you:

- Attract customers to inbox directly
- Set up automatic greetings
- Configure suggested questions for customers
- Track conversion rate from messages

---

## How to Use

### Step 1: Send Image and Information

Send ad image with information in this format:

```
1: Campaign name: skin treatment $99
2: Age: 20-55 years old
3: Gender: Female
4: Lifetime budget: $80
5: Duration: from 12/15 to 1/15
6: Schedule: 7am-11am, 2pm-5pm, 8pm-10pm
7: Location: United States
```

### Step 2: AI Processing

AI will:
- Analyze ad information
- Upload media to Facebook
- Prepare settings

### Step 3: View Confirmation

AI displays complete information:

| Item | Details |
|------|---------|
| Media | Image uploaded successfully |
| Information | Name, budget, age, gender, duration |
| Location | Target country/city |
| Content | Ad caption |
| Message | Greeting and suggested questions |

### Step 4: Confirm and Create

Click **Confirm & Create** to publish your ad.

---

## Message Settings

### Greeting

```
Hi there! How can I help you today?
```

### Ice Breakers (Suggested Questions)

Maximum 3 questions:
1. "I want to book an appointment"
2. "What services do you offer?"
3. "What are your prices?"

---

:::tip Effective Tips
- Use eye-catching images/videos
- Friendly, professional greetings
- Suggested questions that match customer psychology
:::

:::info Best Schedule
Choose hours when customers are most online: 7am-11am, 2pm-5pm, 8pm-10pm
:::
