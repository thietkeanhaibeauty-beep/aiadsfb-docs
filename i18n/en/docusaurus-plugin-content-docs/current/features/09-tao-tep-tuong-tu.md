# Create Lookalike Audience

Create Lookalike Audience from Custom Audience to expand reach.

---

## Description

Lookalike Audience helps you:

- Find people similar to existing customers
- Expand reach with quality audiences
- Improve cold audience ad effectiveness

---

## How to Create

### Step 1: Send Command

```
Create lookalike audience
```

Or combined:

```
Create messengers audience and lookalike
```

### Step 2: Choose Source Audience

Select Custom Audience as source:
- Page engagers
- Messengers
- Video viewers
- Customers

### Step 3: Choose Percentage

| Percentage | Characteristics |
|------------|-----------------|
| 1% | Most similar, smallest reach |
| 2% | Balance between similarity and reach |
| 3% | Larger reach, less similar |
| 5% | Large reach, diverse |
| 10% | Maximum reach |

### Step 4: Choose Country

Select target country (usually United States or your market).

### Step 5: Confirm

AI creates the Lookalike Audience.

---

## Example

```
You: Create lookalike from messengers

AI: Creating Lookalike Audience...

Source: Messengers 30 days (5,234 people)
Percentage: 1%
Country: United States

Created successfully!
Name: LAL 1% - Messengers 30 days
Estimated size: 980,000 people
```

---

:::tip Best Practice
- Source audience should have at least 1,000 people
- Start with 1% to test
- Gradually expand to 2%, 3% if effective
:::
