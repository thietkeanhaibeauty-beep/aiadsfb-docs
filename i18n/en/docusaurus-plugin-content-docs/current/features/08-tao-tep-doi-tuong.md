# Create Custom Audience

Create Custom Audience from Page engagers and run ads to that audience.

---

## Description

Custom Audience helps you:

- Target people who interacted with your Page
- Retarget potential customers
- Increase conversion rate

---

## Audience Types

| Type | Description |
|------|-------------|
| Page Engagement | People who interacted with Page (like, comment, share) |
| Video Views | People who watched videos |
| Lead Form | People who filled out lead forms |
| Message Engagement | People who messaged your Page |
| Website Visitors | Website visitors (requires Pixel) |

---

## How to Create

### Step 1: Send Command

```
Create custom audience
```

### Step 2: Choose Interaction Type

AI will ask you to choose:
- Page Engagement
- Video Views
- Lead Form
- Message Engagement

### Step 3: Choose Time Period

Select interaction timeframe:
- 30 days
- 60 days
- 90 days
- 180 days
- 365 days

### Step 4: Name It

Name your audience, e.g.: "30 Day Engagers"

### Step 5: Confirm

Click confirm to create the audience.

---

## Run Ads to Audience

After creating audiences, you can:

```
Run ad to audience [Audience Name]
```

> See details: [Run Ads to Custom Audience](/features/chay-qc-tep-doi-tuong)

---

:::info Processing Time
Facebook needs 24-48 hours to fully populate Custom Audience.
:::

:::tip Pro Tip
Combine multiple audience types to increase reach and retargeting effectiveness!
:::
