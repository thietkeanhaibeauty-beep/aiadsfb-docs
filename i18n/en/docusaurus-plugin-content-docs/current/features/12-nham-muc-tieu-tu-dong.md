# Auto Targeting Suggestions

AI analyzes services and suggests optimal targeting for Spa/Clinic industry.

---

## Description

AI is specially trained for Spa/Clinic industry to:

- Suggest suitable age ranges
- Recommend relevant interests
- Choose optimal geographic locations
- Evaluate target income levels

---

## Command

```
Suggest targeting for [Service]
```

---

## Example

```
You: Suggest targeting for Spa

AI: Targeting recommendations:

Gender: Female 25-45 years old
Location: NYC, LA
Interests: Beauty, Skincare
Income: Medium+
```
