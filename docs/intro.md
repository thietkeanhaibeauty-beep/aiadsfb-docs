# Giới thiệu AIadsfb.com

Chào mừng bạn đến với **AIadsfb.com** - Nền tảng AI Quảng Cáo Tự Động 24/7 dành riêng cho **Spa/Clinic**.

## AIadsfb.com là gì?

AIadsfb.com là nền tảng giúp bạn tự động hóa hoàn toàn quy trình chạy quảng cáo Facebook Ads thông qua AI. Thay vì phải thao tác thủ công trên Ads Manager, bạn chỉ cần chat với AI như đang nhắn tin với một chuyên gia Marketing.

## Tính năng chính

| Tính năng | Mô tả |
|-----------|-------|
| **Báo cáo ADS tự động** | Xem báo cáo chi tiết theo ngày/tuần/tháng |
| **Quảng cáo bài viết** | Dán link → AI tự tạo quảng cáo |
| **Quảng cáo tin nhắn** | Tạo QC inbox với auto-reply |
| **AI tạo QC 1-click** | Tạo campaign hoàn chỉnh chỉ 1 câu lệnh |
| **Ngân sách tự tối ưu** | AI tự điều chỉnh 24/7 |
| **AI viết content** | Tự động viết caption chuẩn dịch vụ |
| **Quản lý chiến dịch** | Bật/tắt bằng chat hoặc toggle |
| **Tạo tệp đối tượng** | Custom Audience & Lookalike |
| **Mẫu nhắm mục tiêu** | Lưu template targeting |
| **Quy tắc tự động** | Thiết lập automation rules |
